Two steps to your daily paper.

This app fetches live headlines from Reuters, AP & BBC, then uses Google Gemini AI (free) to format them as a broadsheet newspaper — styled like the edition you saw, updated every day in Brisbane time.

Step 1 — Free Gemini API key (2 min):
1. Go to aistudio.google.com/app/apikey
2. Sign in with any Google account
3. Click Create API Key → copy it
Key should start with AIza and be at least 30 characters.
Step 2 — Free Worker URL (10 min):
Follow SETUP.md in the repo to deploy the Cloudflare Worker.
Then paste your Worker URL below.
Leave blank to use the built-in fallback proxy (less reliable).
